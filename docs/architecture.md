# Архитектура инфраструктуры

## Физический хост
- **Proxmox VE 8.x**
- Внешняя сеть: 192.168.163.0/24
- Внутренняя сеть: 192.168.77.0/24

## Виртуальные машины

| Имя | IP | ОС | Роль |
|-----|----|----|------|
| 77gw | 192.168.77.40 | Debian 12 | Шлюз, NAT, HAProxy |
| dc01 | 192.168.77.151 | Windows Server 2025 | DC, DNS |
| zabbix | 192.168.77.132 | Debian 12 | Zabbix + Apache |
| gw01 | 192.168.77.143 | Windows Server 2025 | TS Gateway |
| wsus01 | 192.168.77.144 | Windows Server 2025 | WSUS |

## Сеть и маршрутизация

- **MikroTik**: dst-nat порта 443 → 192.168.163.40 (77gw)
- **77gw**: HAProxy слушает 443, проксирует по SNI
- **DNS split**: внутренний DNS на dc01, публичные записи у регистратора
